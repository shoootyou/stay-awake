Jiggle Mode
  • Power Only — Prevents sleep using system power API. No mouse movement, no permissions needed.
  • Mouse Subtle — Moves the mouse 1 pixel right then back. Barely visible.
  • Mouse Zen — Sends a zero-delta mouse event. Resets idle timer without any visible cursor movement.
  • Mouse Circle — Moves the mouse in a small circular pattern. Most visible but effective everywhere.
